  <h2> Certain States To Be Understood </h2>
  <ul>
  <li> Ready: </li>
  <p> When we can't see the bullet on screen. It is in state of going to be ejected from spaceship.</p>
  <li> Fire: </li>
  <p> The state in which bullet is in moving condition. </p>
  </ul>
